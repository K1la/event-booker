# Веб-интерфейс системы бронирования мероприятий

Этот веб-интерфейс предоставляет две основные страницы для работы с системой бронирования мероприятий.

## Структура файлов

- `index.html` - Главная страница с выбором роли (пользователь/администратор)
- `admin.html` - Панель администратора
- `user.html` - Пользовательская панель
- `styles.css` - Общие стили для всех страниц
- `api.js` - JavaScript библиотека для работы с API

## Функциональность

### Главная страница (`/`)
- Выбор между пользовательской и администраторской панелями
- Красивый интерфейс с карточками ролей

### Панель администратора (`/web/admin.html`)
- **Создание мероприятий**: форма с полями названия, даты/времени и количества мест
- **Просмотр всех мероприятий**: список с детальной информацией
- **Управление бронированиями**: просмотр всех бронирований для каждого мероприятия
- **Отмена бронирований**: возможность отменить любое бронирование

### Пользовательская панель (`/web/user.html`)
- **Просмотр мероприятий**: список доступных мероприятий с информацией о свободных местах
- **Бронирование мест**: форма для бронирования с указанием Telegram ID и количества мест
- **Подтверждение бронирований**: возможность подтвердить бронирование по ID мероприятия
- **Автообновление**: список мероприятий обновляется каждые 30 секунд

## API Endpoints

Веб-интерфейс использует следующие API endpoints:

- `GET /api/events` - получение всех мероприятий
- `GET /api/events/{id}` - получение мероприятия по ID
- `POST /api/events` - создание мероприятия
- `POST /api/events/{id}/book` - бронирование места
- `POST /api/events/{id}/confirm` - подтверждение бронирования
- `POST /api/events/{id}` - отмена бронирования

## Использование

1. Запустите Go сервер
2. Откройте браузер и перейдите на `http://localhost:8080`
3. Выберите роль (пользователь или администратор)
4. Используйте соответствующий интерфейс для работы с системой
